## What is VEIL?

VEIL is a local, client-side privacy layer that provides:

1. **End-to-end encryption** - Messages encrypted before leaving your device
2. **Plausible deniability** - Messages look normal to outside observers
3. **Contextual covers** - Generated messages match conversation context
4. **Zero network footprint** - No servers, no tracking
